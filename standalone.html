ROADMAP
SQLITE_BACKEND_PATCH — server.py
▸ Add these 4 routes to your existing server.py (10 min)
Requires: pip install sqlite-utils — already works with your secured Flask setup. No auth changes needed.
# ── ADD TO server.py (after your existing imports) ──────────────── import sqlite_utils PM_DB_PATH = Path.home() / '.promptmatrix' / 'promptmatrix.db' def get_pm_db(): db = sqlite_utils.Database(PM_DB_PATH) db['pm_store'].create({'key': str, 'value': str}, pk='key', ignore=True) return db # ── 4 routes — add after your existing routes ───────────────────── @app.route('/api/pm-db/all') def pm_db_all(): db = get_pm_db() data = {r['key']: json.loads(r['value']) for r in db['pm_store'].rows} return jsonify({'data': data}) @app.route('/api/pm-db/get') def pm_db_get(): key = request.args.get('key') db = get_pm_db() row = db['pm_store'].get(key) if key else None return jsonify({'value': json.loads(row['value']) if row else None}) @app.route('/api/pm-db/set', methods=['POST']) @require_token def pm_db_set(): d = request.get_json() get_pm_db()['pm_store'].upsert({'key': d['key'], 'value': d['value']}, pk='key') return jsonify({'ok': True}) @app.route('/api/pm-db/del', methods=['POST']) @require_token def pm_db_del(): d = request.get_json() get_pm_db()['pm_store'].delete_where('key = ?', [d['key']]) return jsonify({'ok': True})
DB file: ~/.promptmatrix/promptmatrix.db — auto-created on first write.
/get and /all are open (read-only). /set and /del require your Bearer token (FIX-2 already in server.py).
After adding: restart service → sudo systemctl restart pm-api
